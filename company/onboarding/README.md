# Onboarding

## Purpose

The onboarding directory contains guides and documentation that teach new users and AI assistants how to use the Hackathon Foundation framework effectively.

## Responsibility

This directory bridges the gap between reading about the framework and using it. It provides practical, step-by-step guidance for getting started.

## What belongs here

- Quick-start guides for new users
- AI assistant onboarding guides
- Tutorials for common workflows
- Best practices for using the framework
- Troubleshooting guides

## What does NOT belong here

- Reference documentation (belongs in `docs/`)
- Internal company knowledge (belongs in `company/handbook/`)
- Templates and skills (belongs in `company/assets/`)

## Future roadmap

This directory will be populated with onboarding guides that cover:

1. First-time user setup
2. AI assistant orientation
3. Common workflow tutorials
4. Best practices and tips
5. Troubleshooting common issues
