# Task Assignment

## Purpose

Standard template for assigning a task to a role. The CEO fills this out and provides it to the assigned role before work begins.

---

## Instructions for the assigned role

1. Read `company/handbook/mission.md` and `company/handbook/company-culture.md`.
2. Read your role definition in `company/roles/<department>/<role>.md`.
3. Read the relevant policies listed above.
4. Read `company/workspace/state/current-task.md` and `company/workspace/state/last-action.md`.
5. Read `company/workspace/handoff/resume-protocol.md` if resuming work.
6. Execute following the workflow in `company/assets/workflows/new-task-workflow.md`.
7. Update state files on completion.
