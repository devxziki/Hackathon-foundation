# Security

## Reporting Vulnerabilities

If you discover a security vulnerability in the Hackathon Foundation repository, please report it privately by opening a GitHub issue with the label `security`.

Do not disclose security vulnerabilities in public issues or discussions.

## Scope

This repository contains documentation and structure — not executable code. Security vulnerabilities in this context primarily relate to:

- Misleading or dangerous documentation
- References to unsafe tools or practices
- Exposed credentials or secrets in the repository

## Response

Security reports will be acknowledged within 48 hours and addressed promptly.
