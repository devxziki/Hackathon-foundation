# Integrations

## Purpose

The integrations directory contains guides for using the Hackathon Foundation framework with different AI coding assistants.

## Responsibility

This directory documents how to set up and use each AI coding assistant with the framework. It maps the framework's content (roles, context, rules, skills, templates) to each tool's interface.

## What belongs here

- Integration guides for each supported AI coding assistant
- Tool-specific setup instructions
- Configuration guidance
- Usage patterns and best practices

## What does NOT belong here

- Model information (belongs in `resources/ai/`)
- General documentation (belongs in `docs/`)
- Prompts (belongs in `company/assets/`)

## Future roadmap

This directory will be populated with integration guides for OpenCode, Gemini CLI, Continue, Cline, Roo Code, VS Code Copilot, and other AI coding assistants.
