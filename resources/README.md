# Resources

## Purpose

The resources directory contains public-facing resource catalogs and references that support the Hackathon Foundation framework. These are external resources that users can leverage in their projects.

## Responsibility

This directory curates and catalogs free, high-quality resources that hackathon participants can use — AI models, tools, integrations, examples, and community resources.

## What belongs here

- `ai/` — Free AI model directory and selection guides
- `integrations/` — AI coding assistant integration guides
- `examples/` — Example projects demonstrating framework usage
- `community/` — Community resources and free tool catalogs

## What does NOT belong here

- Internal company assets (belongs in `company/assets/`)
- Public project documentation (belongs in `docs/`)
- Utility scripts (belongs in `scripts/`)

## Future roadmap

This directory will be populated with curated resource catalogs covering AI models, integrations, examples, and community-contributed resources.
