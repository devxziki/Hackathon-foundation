# Examples

## Purpose

The examples directory contains example projects and usage patterns that demonstrate how to use the Hackathon Foundation framework in practice.

## Responsibility

This directory provides reference implementations that users can learn from and adapt to their own projects.

## What belongs here

- Complete example projects (e.g., web app, API service, CLI tool)
- Partial examples demonstrating specific patterns
- Framework usage demonstrations

## What does NOT belong here

- Templates (belongs in `company/assets/`)
- Production project code
- Personal projects

## Future roadmap

This directory will be populated with example projects demonstrating different use cases, project types, and scales of the framework.
