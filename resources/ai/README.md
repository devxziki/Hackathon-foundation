# AI

## Purpose

The AI resource directory catalogs free AI models that can be used with the Hackathon Foundation framework. It provides model profiles, capability comparisons, and selection guidance.

## Responsibility

This directory helps users select and use the right AI model for each task. It documents model strengths, weaknesses, optimal use cases, and recommended engineering roles.

## What belongs here

- Model profile files (one per model)
- Model comparison data
- Model selection guides
- Capability documentation

## What does NOT belong here

- Integration guides (belongs in `resources/integrations/`)
- Prompt templates (belongs in `company/assets/`)
- Workflow definitions (belongs in `company/assets/`)

## Future roadmap

This directory will be populated with profiles for free AI models including DeepSeek, Gemini Flash, Llama, MiMo, Qwen, Nemotron, Gemma, GLM, Kimi, Granite, and Mistral.
